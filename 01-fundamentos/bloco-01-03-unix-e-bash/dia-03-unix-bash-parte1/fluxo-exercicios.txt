Exercício 1

### Para criar uma pasta/ou diretório usamos mkdir ###
    mkdir unix_tests

### Para navegar até a pasta usamos cd + nome da pasta ###
    cd unix_tests

Exercício 2 

### Para criar um arquivo de texto ou qualquer arquivo vazio usamos o touch ###
    touch trybe.text

Exercício 3 

### Para criar uma cópiado arquivo usamos cp + nome desejado ###
    cp trybe.txt trybe_backup.txt
